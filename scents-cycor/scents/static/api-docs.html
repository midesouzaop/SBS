Documentação da API

Endpoints

POST /register

Registra um novo usuário

Body: { "username": "string", "password": "string" }

POST /login

Autenticação do usuário

Body: { "username": "string", "password": "string" }

Retorno: { "token": "string" }

POST /upload

Upload de arquivos de áudio e mídia

Headers: Authorization: token
FormData: audio (file), media (file)

Retorno: { "message": "string", "files": ["string"] }

POST /generate_video

Gera um vídeo a partir dos últimos arquivos enviados

Headers: Authorization: token

Retorno: { "message": "string", "video_url": "string" }

GET /download/<filename>

Download de arquivo

Path: filename (string)

GET /list-uploads

Lista os arquivos enviados

Headers: Authorization: token

Retorno: [{ "filename": "string", "uploaded_at": "string" }]
